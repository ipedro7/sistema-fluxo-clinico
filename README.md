# Sistema de Gestão de Fluxo Clínico

Projeto desenvolvido para a disciplina de Estrutura de Dados I.

O sistema simula um fluxo simples de clínica, usando lista encadeada para cadastro de pacientes, fila para atendimento, pilha para histórico de ações e algoritmos de ordenação para relatórios.

## Estruturas usadas

- Lista Encadeada Simples
- Fila
- Pilha
- Algoritmos de ordenação

## Como executar

No terminal, dentro da pasta do projeto, execute:

```bash
python src/ui/main.py
